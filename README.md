# unlocker
этот анлокер для дбд работает только для эпика и xbox/ms store


Просто запустите программу и подготовтесь к запуску игры.
Как вы будете готовы к запуску игры, то нажмите в проге "СТАРТ"
Подождите пока пройдет загрузка и запускайте игру!
Что работатет и нет?
Работает: Анлок скинов, предметов, аддонов и накрутка престижа.
Не работает: Использование персонажей, которых нет на аккаунте.


После игры обновите кровавую паутину, если у вас пропали перки и предметы.

всё используется через фиддлер и бан получить за него критически мал!
